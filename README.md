# syslock
syslock and sysunlock script for managing filesystem immutable flags.

Now also supports append-only flags (system and user for BSD, and +a/-a for Linux).

This is version 1.15a of 16 April 2026 from https://www.discord.org/lippard/software/

syslock-1.15a.tgz is a Legion of Dynamic Discord signify-signed OpenBSD package.
Signify public key is https://www.discord.org/lippard/software/discord.org-2026-pkg.pub
